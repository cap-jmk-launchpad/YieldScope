# Demo B-roll (Remotion)

Two silent, seamless **10s** loops (1920×1080 · 30fps · H.264) in YieldScope “Ledger at dusk.”

| File | Concept | Wave design |
|------|---------|-------------|
| `scattered-ledger.mp4` | Streams → ledger | **Particles** on multi-layer ocean paths → block ticks |
| `source-weave.mp4` | Sources → pulse | **96 vertical bars** sampling swell/sea/chop + crest particles |

Ocean model: layered swell + sea + chop (Gerstner-lite), not a single sine ribbon.

## Preview / render

```bash
pnpm demo:broll:studio
pnpm demo:broll
```
